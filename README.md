# AI ASMR Video Prompt System

A prompt-engineering workflow for generating surreal AI ASMR videos using generative AI tools.

## Overview

This project automates the creative pipeline for AI-generated ASMR content.

The system generates structured prompts for:
- Image generation
- Motion generation
- Material physics simulation
- Camera behavior
- ASMR interactions

## Supported ASMR Types

1. Bed ASMR
2. River / Liquid ASMR
3. Staircase ASMR

## Workflow

1. Generate images using Nano Banana Pro
2. Generate videos using Veo 3.1 Fast / Grok
3. Edit videos in CapCut
4. Publish to YouTube Shorts / TikTok

## Features

- Automated prompt generation
- Dynamic camera descriptions
- Physics-aware material interactions
- Vertical video optimization
- ASMR-focused motion design
- Surreal environment generation

## Tools

- Nano Banana Pro
- Veo 3.1 Fast
- GeminiGen
- Grok
- CapCut

## Goal

To explore AI-assisted short-form content creation and prompt-engineering workflows for generative media.
