# AI-ASMR-Video-Prompt-System
This project is a full automated prompt generation system for AI ASMR video creation.

The script generates highly detailed prompts for 3 different types of ASMR content. Users first create images using Nano Banana Pro, then generate videos from those images using Veo 3.1 Fast.

The workflow automates most of the creative pipeline, including scene concepts, material physics, camera behavior, motion descriptions, and ASMR-focused interactions. The only remaining manual steps are video editing and publishing.

The project focuses on surreal AI-generated ASMR content optimized for short-form platforms like YouTube Shorts and TikTok.
I actively experiment with AI video workflows, prompt engineering, and generative media automation.
